SLIM WORKSHOP ONLINE
Version 11, for SLiM 5.0 (updated 11 August 2025 to include PDF lecture slides, otherwise unchanged)

Ben Haller
11 August 2025


========================================================


This folder contains online materials for the SLiM Workshop, for those who want to do it themselves rather than attending an in-person workshop.  There are two types of files provided:

    *.mp4 : Video files providing the lectures
    *.pdf : PDF files for the worksheets, to be done after each lecture

Note that worksheets #7, #13, #20, and #22 have no corresponding lecture; this is intentional.

Slides for all of the lectures are provided in the "Lecture slides" folder.

The solutions for each step of the worksheets are provided in SLiM_Workshop_SOLUTIONS_v11.zip.


The home page for the SLiM Workshop is:

    http://benhaller.com/workshops/workshops.html

That page provides more detailed information, including links to software installation instructions, in-person workshops that are scheduled, information on how to get help, and so forth.  It is recommended that you read it.

Happy modeling!
